# India-EXINUS-Forecasting
# Obesity-Levels-Classification
This repository contains information and material for the India EXINUS Forecasting, a personal project. The purpose of the project is to make forecasts for the Indian Rupees to U.S. Dollar Spot Exchange Rate. The project was written in Python. Main tasks where time series analysis, decomposition, data splitting, data scaling and forecasting (with ARIMA, RNN, LSTM, Prophet, woth Roll Forward Cross Validation and evaluation with metrics like MSE, AME, MAPE, MSR, SSE, SSR, SMAPE).
